## Usage

### Prerequisite

Set your github properties to your `local.properties`

```sh
github.token="<YOUR_GITHUB_TOKEN>" # See https://github.com/settings/tokens
github.user="<YOUR_GITHUB_ID>"
```

### Test

```sh
$ ./gradlew check
```
